# ConnectPro - LinkedIn-like Professional Networking Website

This is a static LinkedIn-inspired professional networking website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Header Navigation**: Top navigation bar with logo and menu options
- **Profile Sidebar**: User profile card with stats and connection count
- **News Feed**: Compose posts and view posts from connections
- **Post Interactions**: Like, comment, and share buttons
- **Network Suggestions**: Recommended connections to add
- **Job Recommendations**: Suggested jobs in the sidebar
- **Trending Topics**: Trending hashtags in the right sidebar
- **Profile Modal**: Click profile button to view/edit profile

## File Structure

```
LinkedInClone/
├── index.html    # Main HTML file
├── style.css     # Stylesheet (responsive, modern design)
├── script.js     # JavaScript for interactivity
└── README.md     # This file
```

## How to Run

Simply open `index.html` in any modern web browser. No server or dependencies required.

## Customization

To customize the website:
1. Replace placeholder images with actual profile/user photos
2. Update text content with real names, companies, and job titles
3. Add more posts by duplicating the `.post-card` div structure
4. Modify colors in `style.css` to match your brand

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Responsive Media Queries)
- JavaScript (ES6)
- Font Awesome icons (CDN)

## Browser Support

Works in all modern browsers: Chrome, Firefox, Safari, Edge
