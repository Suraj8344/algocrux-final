Jump Game — Can You Reach the End?

Track the farthest reachable index. If you can reach or pass the last index, return true. O(N).

LC
Practice on LeetCode 55: Jump Game
Open Problem

Greedy Idea

Maintain maxReach = farthest index reachable so far. At each index i, if i > maxReach you're stuck. Otherwise update maxReach = max(maxReach, i + nums[i]).

Algorithm

maxReach = 0
for i = 0 to n-1:
  if i > maxReach: // stuck
    return false
  maxReach = max(maxReach, i + nums[i])
return true

C++ — Jump Game
bool canJump(vector<int>& nums) {
    int maxReach = 0;
    for (int i = 0; i < nums.size(); i++) {
        if (i > maxReach) return false;
        maxReach = max(maxReach, i + nums[i]);
    }
    return true;
}
