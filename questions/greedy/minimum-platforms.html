Minimum Platforms / Meeting Rooms

Find the minimum number of platforms (or rooms) needed so no two overlapping events conflict. O(N log N).

LC
Practice on LeetCode 253: Meeting Rooms II

Key Insight

Create arrival and departure events. Sort all events by time. Sweep through: arrivals need a platform (+1), departures free one (-1). The peak concurrent count is the answer.

Algorithm

events = merge(arrivals as +1, departures as -1)
sort events by time
platforms = 0, maxPlatforms = 0
for each event:
  platforms += event.type
  maxPlatforms = max(maxPlatforms, platforms)

C++ — Minimum Platforms
int minPlatforms(vector<int>& arr, vector<int>& dep) {
    sort(arr.begin(), arr.end());
    sort(dep.begin(), dep.end());

    int plat = 0, maxPlat = 0;
    int i = 0, j = 0, n = arr.size();

    while (i < n && j < n) {
        if (arr[i] <= dep[j]) {
            plat++; i++;          // arrival
            maxPlat = max(maxPlat, plat);
        } else {
            plat--; j++;          // departure
        }
    }
    return maxPlat;
}
