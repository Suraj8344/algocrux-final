Non-overlapping Intervals — Minimum Removals to Eliminate Overlaps

Find the minimum number of intervals to remove so no two intervals overlap. O(N log N).

Key Insight

This is the flip side of Activity Selection. Max non-overlapping = N, so removals = N - maxKept. Sort by end time, greedily keep intervals that don't overlap, count the rest as removals.

The Algorithm

sort by end time
kept = 1, lastEnd = intervals[0].end
for each remaining interval:
  if start >= lastEnd: keep it
  else: remove it (overlap)
return N - kept

C++ — Non-overlapping Intervals
int eraseOverlapIntervals(vector<vector<int>>& intervals) {
    sort(intervals.begin(), intervals.end(),
         [](auto& a, auto& b){ return a[1] < b[1]; });

    int kept = 1, lastEnd = intervals[0][1];

    for (int i = 1; i < intervals.size(); i++) {
        if (intervals[i][0] >= lastEnd) {
            kept++;
            lastEnd = intervals[i][1];
        }
    }
    return intervals.size() - kept;
}
