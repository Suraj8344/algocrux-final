Bubble Sort — Swap Adjacent, Bubble Up

Repeatedly compare adjacent elements and swap if out of order. Largest elements "bubble" to the end each pass.

How it works

Walk through the array comparing adjacent pairs. If left > right, swap them. Each pass pushes the next largest element to its final position. Repeat until no swaps are needed.

Optimization

// Early termination
if no swaps in a pass:
  array is sorted, stop early
// Each pass shrinks unsorted region by 1

C++ — Bubble Sort
void bubbleSort(vector<int>& arr) {
    int n = arr.size();
    for (int i = 0; i < n - 1; i++) {
        bool swapped = false;
        for (int j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                swap(arr[j], arr[j + 1]);
                swapped = true;
            }
        }
        if (!swapped) break; // optimized
    }
}
