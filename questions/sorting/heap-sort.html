Heap Sort — Build, Extract, Repeat

Build a max heap, then extract the maximum repeatedly. O(N log N) guaranteed, in-place.

What does it solve?

Sorts an array in-place by treating it as a binary heap. First, build a max heap so the root is the largest element. Then repeatedly swap the root with the last unsorted element and restore the heap property.

Key insight: after each extraction, the heap shrinks by one and the sorted portion at the end grows by one. The heapify operation runs in O(log N), and we do it N times: total O(N log N).

C++ — Heap Sort 
void heapify(vector<int>& arr, int n, int root) {
    int largest = root;
    int left  = 2 * root + 1;
    int right = 2 * root + 2;

    if (left < n && arr[left] > arr[largest])
        largest = left;
    if (right < n && arr[right] > arr[largest])
        largest = right;

    if (largest != root) {
        swap(arr[root], arr[largest]);
        heapify(arr, n, largest);  // recurse down
    }
}

void heapSort(vector<int>& arr) {
    int n = arr.size();

    // Build max heap
    for (int i = n / 2 - 1; i >= 0; i--)
        heapify(arr, n, i);

    // Extract elements one by one
    for (int i = n - 1; i > 0; i--) {
        swap(arr[0], arr[i]);  // move max to end
        heapify(arr, i, 0);   // restore heap
    }
}
