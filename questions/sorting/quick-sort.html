Quick Sort — Partition Around Pivot

Pick a pivot, partition so everything left is smaller and everything right is larger. Recurse. O(N log N) avg.

How it works

Choose a pivot (last element here). Use two pointers: scan left to right. If element ≤ pivot, swap it to the "less than" partition. After scanning, swap pivot to its final position. Recurse on both sides.

for j = low to high-1:
  if arr[j] <= pivot:
    i++, swap(arr[i], arr[j])
swap(arr[i+1], arr[high]) // place pivot

C++ — Quick Sort (Lomuto)
int partition(vector<int>& arr, int low, int high) {
    int pivot = arr[high];
    int i = low - 1;
    for (int j = low; j < high; j++) {
        if (arr[j] <= pivot) {
            i++;
            swap(arr[i], arr[j]);
        }
    }
    swap(arr[i + 1], arr[high]);
    return i + 1;
}

void quickSort(vector<int>& arr, int low, int high) {
    if (low < high) {
        int pi = partition(arr, low, high);
        quickSort(arr, low, pi - 1);
        quickSort(arr, pi + 1, high);
    }
}
